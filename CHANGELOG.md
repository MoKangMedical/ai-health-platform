# 变更日志

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范。

## [1.0.0] - 2025-04-27

### 新增
- 🎤 **多模态健康检测引擎**
  - 语音分析器：59维声学特征提取
  - 面部分析器：6维面部特征提取
  - 综合健康评估：中医体质诊断、营养需求分析

- 💊 **AI营养配方引擎**
  - 知识图谱：营养成分/健康状况/中医体质节点
  - 配方引擎：10种预置成分，智能配方生成
  - 配方调整：基于用户反馈优化

- 🤖 **AI Agent系统**
  - 小和（协调员）：意图识别与路由
  - 小译（翻译）：19种医学术语解释
  - 小结（总结）：健康报告生成
  - 小智（顾问）：5个领域营养建议

- 🌐 **FastAPI后端**
  - 20+ API端点
  - SQLite数据库
  - 用户/检测/配方/订单管理

- 🎨 **前端界面**
  - 产品展示首页（动画/响应式/定价/评价）
  - 控制台界面（6个功能页面）
  - 健康报告详情页
  - 营养配方详情页

- 🧪 **测试覆盖**
  - 37个单元测试
  - 16个API测试
  - 集成测试

- 📦 **部署配置**
  - Docker镜像
  - Docker Compose编排
  - Nginx配置
  - Systemd服务

### 技术栈
- 后端：FastAPI + SQLAlchemy + Pydantic
- AI：PyTorch + Transformers + 自研多模态模型
- 数据库：SQLite (开发) / PostgreSQL (生产)
- 缓存：Redis
- 部署：Docker + Docker Compose

## 版本说明

### 版本号规则
- **主版本号**：不兼容的API修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

### 发布周期
- 主版本：每季度发布
- 次版本：每月发布
- 修订号：按需发布
