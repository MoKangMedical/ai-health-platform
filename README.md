# AI Health Platform

AI-powered health management platform integrating wearable data, medical records, and personalized health insights.

## Features

- Health Data Integration
- AI Risk Assessment
- Personalized Health Plans
- Wearable Device Sync
- Doctor Portal
- Family Sharing

## Quick Start

```bash
# Clone the repository
git clone https://github.com/MoKangMedical/ai-health-platform.git
cd ai-health-platform

# Open in browser
open docs/index.html
```
