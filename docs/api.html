POST /api/v1/users/register

用户注册

请求参数
参数 类型 必填 说明
openid string 必填 用户唯一标识
nickname string 可选 用户昵称
phone string 可选 手机号 
// 请求
curl -X POST http://localhost:8200/api/v1/users/register \
  -d "openid=test_001&nickname=测试用户"

// 响应
{
  "status": "success",
  "user_id": 1,
  "message": "注册成功"
}
POST /api/v1/detect/comprehensive

综合健康检测

请求参数 (multipart/form-data)
参数 类型 必填 说明
user_id integer 必填 用户ID
audio file 可选 音频文件
image file 可选 图片文件 
// 响应
{
  "status": "success",
  "record_id": 1,
  "report": {
    "overall_score": 85.0,
    "health_summary": "您的整体健康状态良好",
    "risks": [...],
    "tcm_diagnosis": {...},
    "nutrient_needs": {...}
  }
}
POST /api/v1/formula/generate

生成营养配方

请求参数
参数 类型 必填 说明
user_id integer 必填 用户ID
health_record_id integer 必填 健康记录ID
questionnaire json 必填 问卷数据
POST /api/v1/agent/chat

与AI助手对话

请求参数
参数 类型 必填 说明
user_id integer 必填 用户ID
message string 必填 用户消息
agent_type string 可选 Agent类型: coordinator/translator/summarizer/advisor 
// 响应
{
  "status": "success",
  "response": "您好！我是小和，您的AI健康协调员。",
  "agent": "coordinator",
  "intent": "greeting",
  "confidence": 0.95
}
