🔄 数据流

1. 用户上传语音/面部/视频数据

2. API网关接收请求，路由到相应处理模块

3. 健康检测引擎提取多模态特征

4. AI分析生成健康评估报告

5. 配方引擎基于知识图谱生成个性化配方

6. Agent系统提供智能咨询服务

7. 结果返回前端展示
